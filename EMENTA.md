# Ementa sugerida

## Módulo 1: Fundamentos do front-end moderno
*Objetivos:* Apresentar as limitações da construção de interfaces apenas com HTML, CSS e JavaScript puros, mostrando porque bibliotecas como React surgiram e como elas ajudam a organizar aplicações front-end modernas.

- Limitações da abordagem tradicional de interfaces
- HTML e CSS: estrutura, estilo e responsividade
- JavaScript no front-end: DOM, eventos e requisições assíncronas com fetch/AJAX

*Competências:* compreender os desafios de manutenção e escalabilidade do front-end tradicional; revisar os fundamentos de HTML, CSS e responsividade; manipular elementos com JavaScript; trabalhar com eventos, DOM e requisições assíncronas; entender o contexto em que o React se torna útil.

## Módulo 2: Setup e fundamentos do React
*Objetivos:* Introduzir o ambiente de desenvolvimento React e os conceitos centrais da biblioteca, permitindo que os alunos comecem a construir interfaces com componentes reutilizáveis.

- Ambiente de desenvolvimento: Node.js, Vite, estrutura de projeto, ESLint e Prettier
- JSX, componentes, props e composição
- Estado, eventos, renderização condicional, listas e keys

*Competências:* configurar um projeto React com ferramentas modernas; entender a estrutura básica de um projeto; criar e reutilizar componentes; trabalhar com JSX, props e composição; lidar com eventos, estado local e renderização dinâmica; renderizar listas de forma adequada com keys.

## Módulo 3: Hooks essenciais e formulários
*Objetivos:* Capacitar os alunos a controlarem dados dinâmicos da interface usando hooks essenciais, além de construir formulários reativos e integrar a aplicação com fontes externas de dados.

- useState na prática e padrões de atualização
- useEffect: efeitos colaterais, dependências, cleanup e integração com APIs
- Formulários controlados, validação simples e experiência do usuário

*Competências:* usar useState para controlar estados da aplicação; aplicar padrões corretos de atualização de estado; usar useEffect para efeitos colaterais e integração com APIs; compreender dependências e limpeza de efeitos; criar formulários controlados; aplicar validações simples e melhorar a experiência do usuário. 

## Módulo 4: Arquitetura, reaproveitamento e qualidade

*Objetivo:* Ensinar os alunos a sair de uma abordagem apenas funcional e passar a estruturar aplicações React de forma organizada, escalável e com melhor manutenção.

- Componentização, organização de pastas e reutilização
- Estado local vs estado global
- Context API e useReducer
- Custom hooks
- Boas práticas e renderização eficiente com React.memo, useMemo e useCallback

*Competências:* organizar componentes e pastas com mais clareza; identificar oportunidades de reutilização; compartilhar estado entre partes da aplicação; utilizar Context API e useReducer em cenários adequados; criar e utilizar custom hooks; aplicar boas práticas de performance; reduzir renderizações desnecessárias e melhorar a legibilidade do código.
